=== Video Carousel Block ===

A custom WordPress block for displaying video carousels with Flickity.

== Requirements ==
- Flickity library (included via CDN or local file)
- WordPress 6.0+

== Features ==
- Video carousel with custom play/pause controls
- Star ratings for each video
- Responsive design
- Custom navigation buttons

== Usage ==
Add the block to any post or page and configure videos through the block editor.

== Notes ==
The flickity.min.js file should be downloaded from https://unpkg.com/flickity@2/dist/flickity.pkgd.min.js
if you want to host it locally instead of using the CDN version.
